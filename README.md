# Mini Editor

An app for testing random small things. Powered by [`monaco editor`](https://www.npmjs.com/package/monaco-editor).

## Current Features

- The editor
- Textmate grammar support
- Page layout
- A preview frame
- Theme selector
- Save project to local device
- Open a project from the local device
